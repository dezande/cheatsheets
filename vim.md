# Vim

## Moving

|Mapping|Summary|
|-------|-------|
|`H`,`M`,`L`|move (__H__)igh, (__M__)iddle, or (__L__)ow within the viewport|
|`C-u`,`C-d`|move (__u__)p or (__d__)own|
|`C-u`,`C-d`|move (__u__)p or (__d__)own|
|`<NN>G`|(__G__)o to line number NN (useful if you have a failing test on a given line)|
